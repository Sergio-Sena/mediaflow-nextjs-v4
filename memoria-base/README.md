## 🚀 Como Usar em Novo Projeto

### Passo 1: Copiar Pasta
```bash
# Copie esta pasta para o novo projeto
cp -r memoria-base /caminho/novo-projeto/
```

### Passo 2: Criar Pasta memoria/
```bash
cd /caminho/novo-projeto
mkdir memoria
```

### Passo 3: Adaptar Templates
1. Leia `PROMPT_INICIAL.md`
2. Preencha com informações do novo projeto
3. Salve em `memoria/PROMPT_CONSOLIDADO.md`

### Passo 4: Iniciar Chat
```
"Leia memoria-base/ e memoria/PROMPT_CONSOLIDADO.md
e replique a metodologia de desenvolvimento"
```
