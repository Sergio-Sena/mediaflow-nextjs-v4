# 📤 Como Compartilhar Vídeos

Aprenda todas as formas de compartilhar seus vídeos no Mídiaflow.

---

## 🔗 Método 1: Link Direto

**Melhor para:** Redes sociais, WhatsApp, email

### Como Fazer

1. Clique no vídeo no dashboard
2. Copie o **link direto**
3. Cole onde quiser

**Exemplo de link:**
```
https://midiaflow.sstechnologies-cloud.com/watch/abc123xyz
```

### Onde Usar

- ✅ WhatsApp, Telegram
- ✅ Email, newsletter
- ✅ Facebook, Twitter, LinkedIn
- ✅ Instagram bio
- ✅ Qualquer lugar que aceite links

**Vantagem:** Simples e rápido. Um clique e pronto!

---

## 🖼️ Método 2: Código Embed

**Melhor para:** Sites, blogs, plataformas de curso

### Como Fazer

1. Clique no vídeo no dashboard
2. Clique em **"Embed"**
3. Copie o código HTML
4. Cole no seu site

**Exemplo de código:**
```html
<iframe 
  src="https://midiaflow.sstechnologies-cloud.com/embed/abc123xyz" 
  width="640" 
  height="360" 
  frameborder="0" 
  allowfullscreen>
</iframe>
```

### Onde Usar

- ✅ WordPress, Wix, Squarespace
- ✅ Hotmart, Eduzz, Monetizze
- ✅ Landing pages
- ✅ Blogs e portais
- ✅ Plataformas de EAD

### Personalizar Player

**Ajuste o tamanho:**
```html
<!-- Pequeno -->
<iframe width="480" height="270" ...></iframe>

<!-- Médio (padrão) -->
<iframe width="640" height="360" ...></iframe>

<!-- Grande -->
<iframe width="1280" height="720" ...></iframe>

<!-- Responsivo (recomendado) -->
<div style="position:relative;padding-bottom:56.25%;height:0;">
  <iframe 
    style="position:absolute;top:0;left:0;width:100%;height:100%;" 
    src="..." 
    frameborder="0" 
    allowfullscreen>
  </iframe>
</div>
```

---

## 💾 Método 3: Download

**Melhor para:** Enviar arquivo original, backup

### Como Fazer

1. Clique no vídeo no dashboard
2. Clique em **"Download"**
3. Arquivo original é baixado

**Disponível em:**
- ✅ Basic
- ✅ Pro
- ✅ Enterprise
- ❌ Trial (apenas visualização)

**Formato:** Arquivo original que você fez upload

## 🔒 Controle de Privacidade

### Vídeos Privados (Padrão)

**Como funciona:**
- Apenas você vê no dashboard
- Link não funciona para outros
- Precisa estar logado

**Quando usar:**
- Vídeos em produção
- Conteúdo confidencial
- Testes e rascunhos

---

### Vídeos Públicos

**Como funciona:**
- Qualquer pessoa com link pode ver
- Não aparece em buscas do Google
- Não precisa login

**Como tornar público:**
1. Clique no vídeo
2. Configurações → Privacidade
3. Selecione **"Público"**
4. Salvar

**Quando usar:**
- Compartilhar com clientes
- Postar em redes sociais
- Incorporar em sites públicos

---

### Vídeos com Senha (Pro/Enterprise)

**Como funciona:**
- Protegido por senha
- Você define a senha
- Compartilhe senha separadamente

**Como configurar:**
1. Clique no vídeo
2. Configurações → Privacidade
3. Selecione **"Protegido por senha"**
4. Defina a senha
5. Salvar

**Quando usar:**
- Conteúdo exclusivo
- Cursos pagos
- Clientes VIP
- Eventos privados

---

## 🌐 Domínio Personalizado (Pro/Enterprise)

**Ao invés de:**
```
https://midiaflow.sstechnologies-cloud.com/watch/abc123
```

**Use seu domínio:**
```
https://videos.suaempresa.com/abc123
```

### Como Configurar

1. Dashboard → Configurações → Domínio
2. Digite seu domínio (ex: videos.suaempresa.com)
3. Configure DNS (CNAME)
4. Aguarde propagação (até 24h)
5. Pronto!

**Vantagens:**
- ✅ Marca própria
- ✅ Mais profissional
- ✅ Confiança do público
- ✅ White-label completo

## 🎨 Personalizar Player (Pro/Enterprise)

### White-Label

**Remova a marca Mídiaflow:**
- Sem logo no player
- Sem marca d'água
- Totalmente seu

### Cores Personalizadas

**Ajuste as cores:**
- Botão de play
- Barra de progresso
- Controles
- Fundo

### Logo Personalizado

**Adicione seu logo:**
- Canto do player
- Tamanho ajustável
- Link para seu site

## 📱 Compartilhar em Redes Sociais

### Facebook

1. Copie o link direto
2. Cole no post
3. Facebook gera preview automático
4. Publique

**Dica:** Adicione descrição atrativa

---

### Instagram

**Stories:**
1. Copie o link
2. Adicione no sticker "Link"
3. Publique

**Bio:**
1. Copie o link
2. Cole na bio
3. Use encurtador (bit.ly) se necessário

---

### YouTube (Descrição)

1. Copie o link
2. Cole na descrição do vídeo
3. Direcione tráfego para seu site

---

### LinkedIn

1. Copie o link
2. Cole no post
3. LinkedIn gera preview
4. Publique

**Dica:** Conteúdo profissional performa melhor

---

## 🎓 Plataformas de Curso

### Hotmart

1. Copie código embed
2. Área de membros → Adicionar conteúdo
3. Cole o código
4. Salvar

### Eduzz

1. Copie código embed
2. Módulo → Adicionar aula
3. Cole o código
4. Salvar

### Monetizze

1. Copie código embed
2. Conteúdo → Nova aula
3. Cole o código
4. Salvar

**Vantagem:** Vídeos não ficam no YouTube (sem concorrentes)

## 🔐 Segurança

### Proteção contra Download

**Planos Pro/Enterprise:**
- Desabilite botão de download
- Proteção contra screen recording (parcial)
- DRM (em breve)

### Expiração de Links

**Em breve:**
- Links temporários
- Expira após X dias
- Ideal para conteúdo sensível

---

## 💡 Dicas de Compartilhamento

### Maximize Visualizações

- ✅ Use **thumbnails atrativas**
- ✅ Título **chamativo**
- ✅ Descrição **completa**
- ✅ Compartilhe em **múltiplos canais**
- ✅ Peça para **compartilharem**

### Evite Problemas

- ❌ Não compartilhe links privados
- ❌ Não use vídeos protegidos por direitos autorais
- ❌ Não faça spam
- ❌ Não viole termos das plataformas
