# 🔒 Política de Privacidade

**Última atualização: 30 de janeiro de 2025**

Sua privacidade é importante. Esta política explica como coletamos, usamos e protegemos seus dados, em conformidade com a LGPD (Lei Geral de Proteção de Dados).

---

## 1. Quais dados coletamos

### Dados que você nos fornece
- **Nome completo** - para identificação
- **Email** - para login e comunicação
- **Senha** - criptografada, nunca armazenamos em texto puro
- **Telefone** (opcional) - para suporte e 2FA
- **Dados de pagamento** - processados pelo Stripe (não armazenamos cartões)

### Dados coletados automaticamente
- **Endereço IP** - para segurança e localização
- **Navegador e dispositivo** - para compatibilidade
- **Logs de acesso** - para segurança e troubleshooting
- **Cookies** - para manter você logado

### Dados de uso
- **Vídeos enviados** - metadados (nome, tamanho, duração)
- **Visualizações** - quantas pessoas assistiram
- **Tempo de uso** - para melhorar o serviço
- **Features usadas** - para priorizar melhorias

---

## 2. Como usamos seus dados

### Para prestar o serviço
- Processar uploads e conversões de vídeo
- Entregar vídeos via CDN global
- Autenticar seu login
- Fornecer suporte técnico

### Para melhorar
- Analisar uso para identificar problemas
- Estatísticas agregadas (sem identificação pessoal)
- Testes A/B de novas funcionalidades

### Para comunicar
- **Emails transacionais**: confirmação de cadastro, alertas de limite
- **Emails de marketing**: novidades e promoções (você pode desativar)
- **Notificações de serviço**: manutenções, problemas

### Para cumprir a lei
- Obrigações fiscais e contábeis
- Prevenção de fraudes
- Atender solicitações judiciais

- **AWS (Amazon)**: Hospedagem e CDN - [DPA assinado]

---

## 4. Como protegemos seus dados

### Segurança técnica
- **Criptografia em repouso**: AES-256 (padrão bancário)
- **Criptografia em trânsito**: TLS 1.3 (HTTPS em tudo)
- **Senhas**: Hash SHA-256 (impossível reverter)
- **2FA obrigatório**: Camada extra de segurança
- **Backup automático**: A cada 6 horas

### Onde ficam os dados
- **Servidores principais**: AWS us-east-1 (Virgínia, EUA)
- **Backup**: Múltiplas regiões AWS
- **CDN**: 400+ localizações globais

### Por quanto tempo guardamos
- **Conta ativa**: Enquanto você usar o serviço
- **Após cancelamento**: 30 dias (para recuperação)
- **Após 120 dias inativo**: Exclusão permanente
- **Logs de acesso**: 90 dias
