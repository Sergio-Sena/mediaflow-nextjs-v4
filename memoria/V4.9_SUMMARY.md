## 🔍 Exemplos de Uso

### Query CloudWatch Logs Insights

**Todos os erros nas últimas 24h:**
```
fields @timestamp, lambda, message, error
| filter level = "ERROR"
| sort @timestamp desc
```

**Logins por usuário:**
```
fields @timestamp, email, role
| filter lambda = "auth-handler" and message = "Login successful"
| stats count() by email
```

**Latência média por Lambda:**
```
fields @timestamp, lambda, duration_ms
| stats avg(duration_ms) as avg_latency by lambda
```

## 📝 Lições Aprendidas

### 1. MFA Bloqueou CI/CD
**Problema**: GitHub Actions requer MFA para deploy automático.  
**Solução**: Workflows criados e documentados para uso futuro.  
**Aprendizado**: Sempre verificar requisitos de segurança antes de planejar automação.

**Problema**: 2 Lambdas sem prefixo `mediaflow-`.  
**Solução**: Script separado para deploy dessas Lambdas.  
**Aprendizado**: Padronização de nomenclatura é crítica.
